# Härtefallpaket Fahrverbot

Plugin: **VerkehrsOWi-Verteidiger**

## Zweck

Diese Vorlage ist ein freistehendes Arbeitsblatt. Sie darf direkt in eine Mandatsakte kopiert, im Chat ausgefüllt oder als Checkliste für eine Besprechung verwendet werden.

## Pflichtfelder

- Arbeitsplatz, Selbstständigkeit, Pflege, Kinder, Krankheit, Nachweise, Alternativen.
- Vortrag darf nicht überziehen; Glaubhaftigkeit zuerst.

| Feld | Eintrag | Quelle oder Beleg | Status |
| --- | --- | --- | --- |
| Mandant |  |  | offen |

| Frist |  |  | offen |
| Sachstand |  |  | offen |

## Qualitätstor

- Frist geprüft: ja/nein
- Mandantenfreigabe vorhanden: ja/nein
- Belege vollständig: ja/nein
- Offene Annahmen benannt: ja/nein
- Versand- oder Terminrisiko geprüft: ja/nein
